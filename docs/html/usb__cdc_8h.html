Detailed Description

USB CDC driver/wrapper.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

This is essentially a wrapper for the app_usb_cdc_acm library; however, it also manages the file driver for the serial operations, which, in turn, gives the user some distance from the particulars of the transactions.
