Detailed Description

Audio driver module.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

This module contains an audio driver that abstracts from the I2S peripheral. * There are two audio buffers, with their own AHB "secondary" controller (formerly using the problemtic term "slave" but I still feel obligated to mention it here incase you don't know what I'm talking about by "secondary"). Giving them their own AHB means we don't have to worry about bus contention, which was very much an issue during development. - W.J.
