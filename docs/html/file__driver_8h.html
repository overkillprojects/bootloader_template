Detailed Description

File system driver.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

Stix uses an ad hoc filesystem to store its audio files to the flash memory IC.

  • The location and size of the audio sample (wav file without header) is stored at 0xXY where X = activity number, Y = track in activity.
  • Files are stored without fragmentation and end on sector boundaries with FLASH_MEM_SECTOR_SIZE.
