Detailed Description

Activity driver module.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

This module contains a generic activity driver. The idea is to pass to this driver the type of activity you wish to run (balance, etc.) and it will do all the heavy lifting for you, with only some of the callbacks handled in main.
