Detailed Description

Activity Score Queue module.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

Because we have an offline mode, scores need to queue, then transmit. This also means that if the queue is full, the user needs to be aware that they will likely lose score data if they don't sync with the host app.
