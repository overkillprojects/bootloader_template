Detailed Description

LED control driver.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

Abstracting away from the PWM. Note that I didn't bother to use the nrfx_pwm driver. I find it easier with PWM to just use the registers. This is generally true of most of the subsystems (esp. GPIO, PPI, even ADC) but sometimes leveraging nrfx either makes for more readable code or gets you a few helper functions that are nice to have. - W.J.
