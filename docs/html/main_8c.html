Detailed Description

Stix Mindfulness main apllication file.

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

The application takes the user through a sequence of exercises to help them increase focus and mindfulness. The activities are comprised of audio files interspersed with IMU data that is read and analyzed to compute a score. The score is then transmitted to the host via BLE, if available, or stored until it can be transmitted. Code upgrades (DFU) are completed via USB DFU and audio file upgrades are completed via USB CDC.
