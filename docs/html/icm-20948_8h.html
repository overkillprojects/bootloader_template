Detailed Description

Wrapper for the TDK ICM-20948 vendor drivers (Abandon hope, ye who enter here)

Author
Walter Jacob - Overkill Projects, LLC.
Date
2021

An attempt at nicely re-packaging the appalling drivers provided by TDK. N.B. - At some point, someone should probably go through and try to entirely rewrite the TDK drivers. My opinion (W.J.) is that it would make sense to track down the lowest-level functions first and rewrite them one at a time, testing as you go. It will take forever, but at least you could document the functionality. As of now (early 2021) there is no documentation for the DMP subsystem.
