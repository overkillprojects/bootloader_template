Detailed Description

Split unsigned integer unions.

These unions are substitutes for the typical >> 8*n bit shifts. They are a little more readable and clearly express your intent. If you compile with no optimizations (-O0 or -Og, etc.) they are smaller and faster than bit shift, but for most compilers, they produce the same output with any optimization flags enabled.
