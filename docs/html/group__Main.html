◆ ble_rtc_handler()

void ble_rtc_handler ( uint16_t  conn_handle,
ble_rtc_t *  p_rtc,
ble_gatts_evt_write_t const *  p_evt_write 
)

BLE RTC service GATT write callback.

Set local RTC datetime to the value written to the characteristic. We have no true RTC module, so we have to get the datetime from the host. It's up to the host to make sure that we don't drift unacceptably, and to update the datetime if we do.

Warning
We get the datetime from host as 32 bit little endian. The data array from the GATT write event is just uint8_t, so we cast as pointer to 32 bit and dereference to get the value.
Parameters
[in]conn_handleConnection handle of host.
[in]p_rtcBLE RTC service instance.
[in]p_evt_writeGATT write event.
